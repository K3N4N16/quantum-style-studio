# 💎 K3N4N QUANTUM STYLE STUDIO V20

**Quantum Style Studio**, Groq Cloud (Llama 3.3 70B) altyapısını kullanan, web geliştiriciler ve radyo yayıncıları için tasarlanmış yapay zeka destekli bir **CSS & UI Tasarım Laboratuvarı**dır.

## 🚀 Özellikler
- **AI Design Engine:** Doğal dille tarif edilen tasarımları anında HTML/CSS koduna dönüştürür.
- **Cyberpunk UI:** Fütüristik ve karanlık mod odaklı kullanıcı arayüzü.
- **Dynamic Previews:** Google Fonts ve CSS Keyframes desteğiyle canlı önizleme.
- **Quantum Chaos:** Tasarım parametrelerini rastgele savuran yaratıcı mod.

## 🛠️ Kurulum
1. Bu depoyu klonlayın.
2. `pip install -r requirements.txt` ile bağımlılıkları kurun.
3. `.streamlit/secrets.toml` içine Groq API anahtarınızı ekleyin.
4. `streamlit run main.py` komutuyla yayını başlatın.
